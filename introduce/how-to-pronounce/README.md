## Easy Mode
My name can be seperated into three korean syllables. which sounds like following:  
```
박 (Park - The same pronunciation with "Park" that you walk in)
상 (Sang - simillar pronunciation with Past tense of sing)
희 (Hee  - Try to pronounce 'ee' in cheese but instead start with 'h')
```

> **Small Note:**  
> I usually use `hui` instead of `hee`. but this is because my legal name defined by S.Korean Government is using `hui` instead.  
> due to 1:1 syllable reference table that National Institute of Korean Language created which doesn't consider any real-life pronunciation. 
>

## FYI: East Asian Name Structure
Since I was born in Korea, My name scheme is strictly affected by East Asian name structure.  
Strangely though, East Asian name structure looks like following:

```
{{Family Name}} {{Given Name}}
```

As you can see, "First Name" doesn't come first in East Asian name structure.
